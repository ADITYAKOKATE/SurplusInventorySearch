# Surplus Inventory Search

This repository contains the full stack for the Surplus Inventory Search project, including a Database API and a Search Frontend/Backend.

## Project Structure

- `inventory-database-project/`: Backend API for managing suppliers and inventory data.
- `inventory-search-project/`: Search-focused frontend and backend for finding and displaying surplus inventory.
